# 北京大学山地越野赛网站

这是为 `pku-ultra-running.com` 准备的纯静态网站。直接部署仓库根目录即可，入口文件是 `index.html`。

## 文件结构

- `index.html`：首页内容、SEO 与公开资料来源链接
- `styles.css`：响应式视觉样式
- `script.js`：移动端导航、高亮导航、图库预览
- `assets/photos/`：部署用赛事照片
- `CNAME`：GitHub Pages 自定义域名
- `robots.txt`、`sitemap.xml`：搜索引擎基础文件

正式报名、竞赛规程、轨迹文件、成绩查询等信息发布后，可优先更新 `index.html` 中的 `#notice`、`#race` 与页脚资料来源。
